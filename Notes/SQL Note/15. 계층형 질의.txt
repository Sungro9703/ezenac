1. 오라클 계층형 질의 (Hierarchical Query)
    1) 테이블에 계층형 데이터가 존재하는 경우
       데이터를 조회하기 위해 계층형 질의를 사용
    2) 계층형 데이터란 동일 테이블에 계층적으로 상위와 하위 데이터가 포함된 데이터를 의미
        - START WITH 조건
            - 계층 구조 전개의 시작위치를 지정하는 구문
            - 루트 데이터를 지정
        - CONNECT BY [PRIOR] 
            - 다음에 전개될 자식 데이터를 지정하는 구문

    3) 가상 컬럼
        - LEVEL
            - 루트데이터면 1
            - 그 하위 데이터면 2
            - 하위데이터가 있을 때마다 1씩 증가
        - CONNECT_BY_ISLEAF
            - 전개과전ㅇ에서 해당 데이터가 리프 데이터면 1, 그렇지 않으면 0

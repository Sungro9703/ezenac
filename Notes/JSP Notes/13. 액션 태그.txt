1. 액션 태그 (Action Tag)
    1) JSP 표준 태그
    2) 페이지 사이에서 이동을 제어하거나
       자바빈을 생성할 때 주로 사용
    3) 특별한 선언없이 <jsp:태그면 /> 형태로 사용
    4) 태그처럼 사용하지만 그 뒤에는 JSP가 수행됨
        -WAS에서 처리된 후 결과만 출력

2. 액션 태그 특징
    1) XML 문법 따름
    2) 반드시 종료 태그를 사용
    3) 액션 태그 사이에 주석을 사용하면 에러 발생
    4) 액션 태그에 속성값을 부여할 때에는 표현식 <%= %>을 사용할 수 있음

3. 용도에 따른 구분
    <jsp:include></jsp:include>
    <jsp:forward></jsp:forward>
    <jsp:useBean></jsp:useBean>, <jsp:setProperty></jsp:setProperty>, <jsp:getProperty></jsp:getProperty>
    <jsp:param></jsp:param>

4. <jsp:include>
    1) 외부 JSP 파일을 현재 JSP 파일로 포함시키는 기능

            include 지시어                                  <jsp:include> 액션 태그
===============================================================================================
형식        <%@ include file="포함할 파일의 경로" %>          <jsp:include apge="포함할 파일의 경로" />

포함방식    페이지 자체를 현재 페이지에 포함시킨 후             실행의 흐름을 포함시킬 페이지로 이동시킨 후
            컴파일 진행                                      실행한 결과를 현재 페이지에 포함시킴

변수        포함시킨 파일에서 생성한 변수 사용 가능             포함시킨 파일에서 생성한 변수 사용 불가

page 영역   공유됨                                           공유되지 않음

request영역 공유됨                                           공유됨

5. <jsp:forward>
    1) 포워드는 현재 페이지에 들어온 요청을 다음 페이지로 보내는 기능
    2) 포워드는 버퍼를 사용
    3) 다음 페이지로 요청을 전달하는 것이 목적
        - 이동된 페이지와 request 영역을 공유
        - URL이 변경되지 않는 특징

6. <jsp:useBean>, <jsp:setProperty>, <jsp:getProperty>
    1) 자바빈즈(JavaBeans)를 생성하거나 설정할 때 사용
    2) 자바빈즈: 데이터를 저장하기 위한 멤버 변수, 게터/세터 메서드로만 이루어진 클래스
        - 기본(default) 패키지 이외의 패키지에 속해 있어야 함
        - 멤버 변수(속성, 프로퍼티)의 접근 지정자는 private로 선언
        - 기본 생성자가 있어야함
        - 멤버 변수에 접근할 수 있는 게터/세터 메서드가 있어야함
        - 게터/세터 메서드의 접근 지정자는 public 이어야 함

    3) 자바빈즈 생성
        - <jsp:useBean id="자바빈즈 이름" class="사용할 패키지와 클래스명" scope="저장될 영역" >
            - id: 자바빈즈 객체의 이름 지정 
            - class: 사용하려는 자바빈즈 객체의 실제 패키지명과 클래스명 지정
            - scope: 자바빈즈가 저장될 내장 객체 영역 지정
                     생략하면 기본값인 page 영역이 지정
        - 자바빈즈 생성할 때는 기본 생성자 호출
            - 기본 생성자 없다면 에러 발생
    
    4) 멤버 변수 값 설정/추출
        - <jsp:setProperty name="자바빈즈 이름" property="속성명(멤버변수)" value="설정값" >
            - name: jsp:useBean>의 id 속성에 지정한 자바빈즈 이름을 지정,
                    인스턴스 변수를 지정하는 것과 동일
            - property: 자바빈즈의 멤버 변수명
            - value: 멤버변수에 설정할 값 지정
        -<jsp:getProperty name="자바빈즈 이름" property="속성명(멤버변수)" >

    5) 와일드카드(*)로 폼값 한번에 설정
        - property 속성에 와일드카드를 사용하면 <form> 태그를 통해 전송되는 모든
          폼값을 한번에 자바빈즈에 입력할 수 있음

7. <jsp:param>
    1) 다른 페이지로 매개변수를 전달
    2) <jsp:include>, <jsp:forward> 액션 태그와 함께 사용
    3) 페이지 사이의 매개변수는 모두 request 영역에 생성

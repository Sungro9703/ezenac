8. 조건문


1. 조건문이란?
    1) 주어진 조건에 따라 다른 실행이 이루어지도록 구현
    2) 조건문은 조건식의 결과가 'true인 경우', 'false인 경우'의 두가지 흐름을 만들어냄
    3) if문, switch문

2. if문 문법
    1) if(조건식) {
        수행문;         // 조건식이 '참'인 경우 수행문이 실행
    }

    2) if ~ else 문
        if(조건식) {
            수행문1;      // 조건식이 '참'인 경우 수행문이 실행
        }
        else {
            수행문2;      // 조건식이 '참'이 아닌 경우 수행문이 실행
        }

    3) 조건이 여러 개 일 때의 if문
        if ~ else if ~ else 문
        - 하나의 상황에 대한 조건이 여러개로 나뉘고
          각 조건에 다른 수행이 이루어져야 할 경우 사용
        - 각 조건은 상호 배타적임.
        
        if(조건식) {
            수행문1;        // 조건식1 '참'인 경우 수행하고 전체 조건문을 빠져나감
        }
        else if(조건식2) {
            수행문2;        // 조건식2 '참'인 경우 수행하고 조건문을 빠져나감
        }
        else if(조건식3) {
            수행문3;        // 조건식3 '참'인 경우 수행하고 조건문을 빠져나감
        } 
        else {
            수행문4;        // 위 조건이 모두 해당되지 않는 경우 수행(디폴트 조건).
        } 

    4) 중첩 if문
        - if문 안에 또 다른 if문을 중첩해서 넣을 수 있다.
        - 중첩 if문 개수는 제한이 없지만 너무 많은 중첩은 가독성이 저해됨.

1. 블록 레벨 요소 & 인라인 레벨 요소
    1) display 속성
        - 기본값이 블록 or 인라인 설정됨

            - 블록 레벨 요소는 요소가 있는 가로 줄 전체를 차지
                - 상자(레이아웃)를 만들기 위한 요소
                - <div></div>
                    - 대표적인 블록 요소
                    - 본질적으로 아무것도 나타나지 않는, 콘텐츠 영역을 설정하는 용도
                    - 특별한 의미가 없는 구분을 위한 요소
                    - 요소가 수직으로 쌓임
                    - 부모 요소의 크기만큼 자동으로 늘어남  (너비)
                    - 포함한 콘텐츠 크기만큼 자동으로 줄어듦 (높이)

            - 인라인 레벨 요소는 요소가 있는 공간만을 차지
                - 글자를 만들기 위한 요소
                - <span></span>
                    - 대표적인 인라인 요소
                    - 본질적으로 아무것도 나타나지 않는, 콘텐츠 영역을 설정하는 용도
                    - 요소가 수평으로 쌓임
                    - 포함한 콘텐츠 크기만큼 자동으로 줄어듦

    2) 인라인 레벨 요소로 바꾸기
    3) 블록 레벨 요소로 바꾸기
    4) 블록과 인라인 존재 이유는 보기 좋게 배치하기 위함

2. 박스 모델 : 테두리, 패딩, 마진
    1) 기본적으로 모든 HTML 요소를 감싸는 상자
    2) 요소, border, padding, margin으로 구성
        - 요소: 테스트, 사진 등 보여줄 대상
        - 패딩: 요소 주변 영역을 감쌈
        - 테두리: 요소와 패딩을 감싸는 테두리
        - 테두리 밖의 영역을 감쌈

3. 반응형 웹
    1) 반응형 디자인
        - 화면 해상도에 따라 HTML과 CSS를 자동으로 맞춰서 보여주는 기능
        - 미디어 쿼리(media query)를 사용하면 됨
            - 미디어 타입과 특징에 따라 다른  CSS 스타일을 적용
            - @media  (속성) {적용 값}
            - @import
                - 문서 최상단에 있어야 함

            -   
                <style>
                    @media 미디어타입 (적용조건) {
                        규칙
                    }
                </style>

                - 미디어타입: all, print, screen, speech

4. float 속성을 이용한 레이아웃
    1) 웹페이지 요소를 공중에 띄워서 화면의 좌측 또는 우측에 배치할 수 있음
    2) float 속성 값
        - left  : 해당 요소를 부모 요소로 만들어 좌측에 배치
        - right : 해당 요소를 부모 요소로 만들어 우측에 배치
    3) clear 속성
        - 하나의 요소에 float 속성이 적용되면
          그 다음에 오는 요소들도 계속 float 속성의 영향을 받음
        - float 속성이 적용된 요소 다음에 오는 요소들 float 속성의 영향을
          받지 않고 새로운 줄에 배치하고자 할 때 사용
        - left  : 이전에 사용된 float: left; 의 기능을 해제
        - right : 이전에 사용된 float: right; 의 기능을 해제
        - both  : 이전에 사용된 float: left; , float: right; 의 기능을 해제

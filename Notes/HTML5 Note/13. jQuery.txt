1. jQuery
    1) DOM의 쉬운 조작, 쉽고 일관된 이벤트 처리, 각종 효과 기능 제공(시각화 지원),
       쉬운 Ajax 지원 등 자바스크립트 전반을 지원하는 라이브러리(프레임워크)
    2) 자바스크립트에서 DOM 객체의 질의(query)해 조작한다는 의미
        => 문서 탐색 및 조작을 위한 쉬운 노드 접근 기술 제공
    3) HTML DOM 객체에 접근하기 위해 셀렉터(selector) 메커니즘을 사용
        - CSS 셀렉터로 문서의 특정 노드를 찾을 수 있음

2. $()
    1) HTML DOM의 특정 노드에 접근하거나, DOM 요소를 생성하거나,
       DOM 구조의 로딩을 완료한 후 실행할 콜백함수를 지정하기 위해 사용하는 함수
    2) ()안에 매개변수로 셀렉터를 기술
        - 셀렉터를 해석해 HTML DOM 구조 상의 노드를 찾아 jQuery 객체로 변환해 반환
        - 해당 객체와 이벤트를 바인딩하거나, 효과를 연결하는 작업 가능
    3) $
        - jQuery의 별칭
    4) jQuery 객체의 메소드는 jQuery 객체 자신을 반환 값으로 전달하기 때문에
       메소드 체인 구성이 가능

3. jQuery(document).ready(function() {...}) ,
   $(document).ready(function() {...})
    1) jQuery 이벤트인 문서 로딩 이벤트에 대한 이벤트 핸들러 구현할 때 사용
        - 매개변수로 전달된 document 객체를 jQuery 객체로 변환하고,
          jQuery 객체의 ready() 메소드의 매개변수인 함수가
          문서 로딩 이벤트의 이벤트 핸들러 함수로 등록됨

4. 셀렉터
    1) 문서 내의 노드를 쉽게 식별하고 참조하기 위해 사용하는 기술
    2) CSS 셀렉터와 동일

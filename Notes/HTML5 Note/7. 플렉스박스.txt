1. 등장 배경
    1) 더 다양한 레이아웃을 만들고자 고안된 기법
    2) flex :유연한
    3) 화면 레이아웃을 유연하게, 자유자재로 배치할 수 있게 해줌
    4) 기존에 사용했던 블록 레벨 요소, 인라인 레벨 요소 방식보다 훨씬 강력하고
       편리한 기능이 많음

2. 플렉스박스로 정렬하기
    1) flexbox 
        - 사용자 인터페이스 디자인에 최적화된 레이아웃을 정의하는 CSS
        - 이미지. 텍스트 같은 요소를 원하는 곳에 배치하는 데 사용
        - 여백을 없앨 뿐 아니라 축을 변경, 자식간의 정렬을 가운데 정렬 등 가능
        - 유연하게 박스 레이아웃을 변경할 수 있음
        - 두축(주축, 교차축)을 기준으로 움직이기 때문에 축 방향을 기준으로 요소를 움직일 수 있음
        - 제 1원칙 : '부모만이 플렉스할 수 있다'
            - 반드시 부모 태그를 선택자로 지정해야 함
            - 자식 요소를 원하는 방향으로 유연하게 배치할 수 있음
    2)
        <style>
            적용 대상 태그
            ----
            body {
                display: flex;
            }
        </style>

    3) 공간에 맞추기 : display flex
    4) 주축 정렬하기
        - justtify-content 속성
            - 웹 페이지의 축을 따라 요소 사이에 공간을 만드는 방법을 정의
            - 축: 주축 교차축
                - 별도의 설정이 없다면 주축은 가로를, 교차축은 세로를 기본값으로 가짐
            - flex-start    : 시작 부분에 정렬
                flex-end      : 끝에 정렬
                center        : 태그의 주축을 중앙에 정렬
                space-between : 요소가 축을 따라 펼쳐짐 (각 요소 사이에 공간이 있음)
                                자식요소가 균등한 간격으로 정렬
                space-around  : 요소가 축을 따라 펼쳐지고 가장자리 주변에도 공간이 있음

    5) 교차축 정렬하기

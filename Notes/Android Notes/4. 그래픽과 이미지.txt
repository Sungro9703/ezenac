1. 안드로이드에서 그래픽 구현
    1) XML
    2) onDraw() 메서드 안에서 draw...()와 같은 메소드를 호출하여 직접 그림

2. Canvas 클래스와 Paint 클래스
    1) Canvas
        - 그림을 그리는 캔버스
        - 도형, 이미지, 텍스트 등을 그리는 다양한 메소드를 가짐
    2) Paint
        - 그림을 그리는 물감과 붓
        - 색상이나 선의 스타일, 채우기 스타일, 폰트 등 그리기 속성을 가짐

3. 패스(Path) 그리기
    1) 복잡한 경로 표현
        - 패스는 직선, 타원, 곡선으로 이루어질 수 있음
        - 캔버스의 메서드 drawPath() 이용

4. 이미지 표시하는 방법
    1) 레이아웃 파일에 이미지를 표시
        - 리소스로 표시
        - 먼저 이미지 파일을 프로젝트의 리소스로 추가
            - res/drawable 폴더에 추가
    2) 캔버스에서 이미지 그려주기
        - 이미지 파일 경로 이용하여 비트맵으로 읽어오는 방법

5. drawable animation
    1) 여러개의 이미지가 순서대로 재생되어 생성
    2) XML 파일 생성
